## Занятие 4 - Принципы SOLID  
### Вариант 2 - Паттерн Наблюдатель  
Напишите свой класс StringBuilder, с возможностью оповещения других объектов об изменении своего  
состояния. Для этого делегируйте все методы стандартному StringBuilder, а в собственном классе  
реализуйте шаблон проектирования «Наблюдатель».  

Требования:  

- Данный класс предоставляет возможность поставить обработчик на событие изменения объекта,  
и таким образом, уведомлять всех наблюдателей о изменении своего состояния
- Добавить простейшую демонстрацию работы с данным классом  
- Отправить ссылку на репозиторий с решением в текстовом файле  

## Реализация  

Для демонстрации аттерна Наблюдатель, необходимо создать 
- класс Subject (Субъект), который генерирует обновления и  
- класс Observer (Наблюдатель), которому эти обновления приходят.

У Субъекта есть методы для подписки, отказа от подписки и оповещения всех своих подписчиков,  
у Наблюдателя — метод, который вызывается при получении новых данных от Субъекта.  

В моей реализации паттерна использованы готовые возможности Java — из пакета java.beans  

Для создания класса MyStringBuilder (Субъект) использован экземпляр класса PropertyChangeSupport.  
Регистрация наблюдателей происходит с помощью метода addPropertyChangeListener.  
Удаление наблюдателей - с помощью removePropertyChangeListener.  
Для оповещения наблюдателей — метод firePropertyChange.  
Он принимает три параметра: тип изменений, предыдущие данные и новые данные.  

Класс MyListener (Наблюдатель) реализует интерфейс PropertyChangeListener  
Реализуемый метод propertyChange получает для обработки событие изменения  
в которое можно передать тип изменения, старое и новое значение.  

В классе Main
1. Инициализируется экземпляр класса MyStringBuilderс помощью строки "Hello"  
2. Инициализируется экземпляр класса MyListener  
3. Класс MyListener регистрируется для прослушивания событий в классе MyStringBuilder  
4. Выполняетс добавление строки " world!"  

В результате в консоль выводится следующее сообщение:  
New subject event << append >>. Old string << Hello >>, new string << Hello world! >>
